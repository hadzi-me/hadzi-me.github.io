Sudoku

I found this code on Github and was very interested in learning how to code a sudoku puzzle. Turns out there is much more work behind my favourite puzzle than I thought. With this simple example I see that it takes a lot of tedious work. While trying to understand it, I already know I would love the code to be able to:

  • to show differences in colours within the inner 3x3 grids better,
  • to only allow numbers to be entered in the cells,
  • to show how many numbers are left,
  • or display feedback for correct or wrong entries.
Anyways, go ahead and give it a try. You can find the code on this website.
